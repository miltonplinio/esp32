# esp32
ESP32 Projects

Here you can find some projects for the IOT module ESP32 that I modified some codes that I found from other people.
The ESP32 used is the one from the picture below. All projecs take advantage of the 0,96" OLED Display with 128Ã—64 resolution.

GPIOs:
D0 - 16
D1 - 5 // I2C Bus SCL (clock)
D2 - 4 // I2C Bus SDA (data)
D3 - 0
D4 - 2 // LED_BUILTIN
D5 - 14 // SPI Bus SCK (clock)
D6 - 12 // SPI Bus MISO
D7 - 13 // SPI Bus MOSI
D8 - 15 // SPI Bus SS (CS)
D9 - 3 // RX0 (serial console)
D10 - 1 // TX0 (serial console)
